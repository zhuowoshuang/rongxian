# 清数智算投资人演示状态说明

## 当前可真实演示的能力
- 产品定位：智能投研工作台 / 研究辅助系统 / 中后台 MVP，不是投顾系统，不是自动交易系统。
- 真实前后端：前端运行在 `http://127.0.0.1:4101`，后端运行在 `http://127.0.0.1:4210`。
- 真实数据库：当前实例使用 SQLite 数据库，页面可见数据库大小约 `413.9 MB`。
- 真实数据规模：
  - 股票 `8314`
  - 行情 `1080820` 左右（以管理中心/数据库为准）
  - 评分 `8314`
  - 信号 `8314`
  - 系统报告 `33`
  - 券商研报索引 `158`
- 真实接口与权限：
  - 未登录访问 `/api/signals` 返回 `401`
  - `analyst` 访问 `/api/admin/system-status` 返回 `403`
  - `admin` 访问 `/api/admin/system-status` 返回 `200`
  - `analyst` 访问 API 配置新增/测试接口返回 `403`
- 真实报告能力：
  - 系统研究报告、个股研究报告、风格研究报告都可真实生成并写入数据库
  - `/api/reports/33/pdf` 返回 `200 application/pdf`
  - HTML 报告详情页可打开，个股报告可按 `report_id` 跳转到正确页面
  - PNG 为当前 HTML 页面快照，不是独立审计报告
- 真实回测与模拟：
  - `/api/backtest/meta?market=A_SHARE` 返回真实日期范围
  - 当前 A 股可回测范围：`2025-12-29` 至 `2026-06-26`
  - 回测页面可展示真实研究测算结果
  - 组合模拟流程可完成“添加持仓 → 开始模拟 → 查看结果”
- 真实管理员后台：
  - 可查看系统健康状态、数据库状态、Redis 状态、数据源模式、默认密码风险
  - 可查看最近报告生成、PDF 导出、回测/模拟、API 配置操作日志摘要
  - API 配置支持新增、编辑、测试、删除，密钥前端保持脱敏
- 真实评分追溯：
  - 个股详情页展示五维评分、数据来源、更新时间、缺失字段
- 真实股票池：
  - 股票池来自后端真实筛选规则，不是前端静态标签

## 演示时应如何介绍
- 可以说：这是一个已经接通真实数据库、真实后端接口、真实报告生成、真实 PDF 导出、真实回测边界和真实管理后台的智能投研工作台 MVP。
- 可以说：系统强调研究辅助、评分追溯、信号解释、股票池规则说明、运维可见性。
- 应明确说：页面里的研究评级、研究仓位、模型观察价、风险警戒价、研究组合表现、历史研究测算都属于研究口径，不构成投资建议。

## 当前仍属于研究 / 模拟口径的部分
- “高关注 / 增强关注 / 观察 / 风险升高 / 回避观察”是研究优先级，不是买卖指令。
- “研究仓位”“模型观察价”“风险警戒价”来自规则推导，不代表实盘执行价格。
- “研究组合表现”“回测结果”“组合模拟结果”属于研究视图 / 非实盘 / 不代表未来收益。
- 券商研报模块展示的是第三方资料索引，不等于平台原生观点。
- Dashboard 在当前数据规模下使用演示缓存展示聚合结果，页面已明确标识“演示缓存 / 非实时”。

## 当前不能对外承诺的能力
- 不能承诺这是投顾系统或自动荐股系统。
- 不能承诺存在自动下单、自动交易、券商交易联动。
- 不能承诺页面展示的是实盘收益、真实账户收益或收益保证。
- 不能承诺所有数据都是生产级实时数据。
- 不能承诺当前版本已完成生产环境安全收口。

## 上线前仍需注意的事项
- 默认密码风险仍存在：开发环境可演示，生产环境必须改为部署时初始化密码。
- Redis 当前为 `不可用`，不影响本地演示，但会影响完整缓存和运维能力。
- 报告正文已经做了展示层合规降级，但个别历史内容仍建议继续做数据库级清洗。
- 若先执行 `npm run build` 再继续使用 `next dev`，可能需要清理前端 `.next` 并重启开发服务，以避免开发态 chunk 缓存异常。

## 推荐演示路径
1. 登录管理员账号，进入策略总览，说明数据截至日期、研究口径和今日判断依据。
2. 进入信号中心，展示研究评级映射、数据来源和关键字段。
3. 进入股票分析和个股详情页，展示五维评分追溯、缺失字段和相关报告。
4. 进入股票池，展示后端真实规则、入池原因和风险标签。
5. 进入报告中心，演示“先选股票、再选风格、生成个股报告、查看 HTML、下载 PDF/PNG”。
6. 进入回测中心，展示真实日期边界、研究测算结果和组合模拟流程。
7. 进入管理中心，展示系统状态、数据规模、最近任务日志和 API 配置能力。
