# 清数智算投资人演示版状态说明

## 当前事实

- 数据库并非空库，真实股票、评分、信号、报告与回测样本已经接通。
- `002415 海康威视` 与 `600519 贵州茅台` 仍然是当前最完整的真实链路演示样本。
- 演示数据默认隔离，正式页面不会自动混入演示评分或演示信号。
- 全站继续保持“智能投研工作台 / 研究辅助系统”口径，不构成投资建议。

## 登录与会话

- 已登录用户再次访问 `/login`，会自动校验会话并跳转到 `/dashboard`。
- 访问根路径 `/` 时：
  - 已登录跳转 `/dashboard`
  - 未登录跳转 `/login`
- 登录页支持 `redirect` 参数，登录成功后优先回到原目标页面。
- 登录过期时，前端会清理本地会话并提示：
  - `登录已过期，请重新登录后查看真实数据。`
- 退出登录时会清理本地 `token`、`user` 以及相关会话状态，不再把受保护页面伪装成空数据。

## 未登录 / 权限不足 / 服务异常

- `401` 不再显示成“暂无数据”，而是明确提示登录过期。
- `403` 不再显示成空页面，而是明确提示当前账号无权访问。
- 网络异常或后端未启动时，前端显示：
  - `后端服务暂不可用，请确认本地后端已启动。`

## 股票搜索口径

### TopSearch

- 中文名称搜索优先走数据库，不依赖 AData：
  - `海康威视 -> 002415`
  - `安克创新 -> 300866`
- 六位代码同样优先走数据库，本地无结果时再尝试补充数据源。
- AData 仅作为补充数据源，不承担中文主搜索。
- 补充数据源失败时，只提示补充数据源异常，不覆盖数据库结果。

### 报告中心

- 报告中心个股搜索同样使用数据库搜索，中文名称可直接命中。

## Signals 页面解释

- Signals 页面中“正式研究信号少”不代表系统没有数据。
- 当前页面会区分：
  - 正式研究信号
  - 风险观察样本
  - 数据质量受限样本
  - 演示信号
- 风险观察不等于正式推荐。
- 数据质量受限不等于看空。
- 演示信号默认隔离，只有用户主动切换后才会展示。

## 股票库口径

- `/stocks` 默认只展示真实评分样本，不自动混入演示评分。
- 页面统计会区分：
  - 真实评分样本
  - 当前筛选结果
  - 当前页展示
  - 正式研究样本
  - 数据质量受限样本
  - 演示评分样本
- 股票库表格已改为投资人友好的简化结构，详细维度收进展开区。

## 个股详情口径

- `ready_full` 显示为“完整链路”。
- `no_data` 显示为“暂无数据”。
- `data_quality_limited` 显示为“数据质量受限”。
- 对于 `603173`、`300866` 这类基础股票已存在但暂无行情 / 财务 / 评分覆盖的样本：
  - 页面显示“数据不足”说明
  - 不再误报为超时或未找到
- `999999`、非法代码等真正不存在的标的，才显示“未找到该股票”。

## 报告口径

报告已按数据状态分层：

1. 完整链路报告
   - 适用于 `002415`、`600519` 等完整样本
   - 包含评分拆解、信号解释、风险提示

2. 评分样本报告
   - 适用于已有评分但链路尚不完整的样本
   - 说明得分项、扣分项、缺失项与为何暂不进入正式信号

3. 数据状态报告
   - 适用于 `603173`、`300866` 等无完整覆盖样本
   - 说明已有哪些数据、缺哪些数据、为什么不能形成真实评分
   - 推荐替代演示样本：
     - `002415 海康威视`
     - `600519 贵州茅台`

## 股票池口径

- 股票池为空不代表系统无数据。
- 当真实评分样本已接入，但没有样本达到池门槛时，页面会明确解释真实原因。
- 页面会同时展示真实评分样本数、正式研究样本数、风险观察样本数、演示评分样本数。

## 回测中心口径

- 回测中心已区分两类能力：
  - 基础行情回测：只要求价格样本充足
  - 完整因子回测：需要财务、技术指标与评分链路支持
- 缺少财务数据时，不再误报为“行情样本不足”。
- `002415`、`600519` 作为推荐样本，可直接用于基础行情回测。
- 页面会明确展示当前股票支持哪种回测模式，以及不可运行时的真实原因。

## 演示数据隔离

- Dashboard、Signals、Stocks、Pools 支持用户主动切换“查看演示数据”。
- 默认仍是 `include_demo=false`。
- 演示数据会明确标记“仅用于功能体验，不代表正式研究结果”。
- 演示数据不进入正式统计口径。

## 研究边界

- 本系统是研究辅助系统，不构成投资建议。
- 页面、报告、评分、信号、股票池、回测结果均属于研究测算 / 非实盘 / 不代表未来收益。
